#  Trip Diary - Expense Splitter (Streamlit App)

A simple and clean **Trip Expense Splitter** web app made with **Streamlit**.  
Add friends and their expenses, then get the exact **who pays whom** settlement.  
You can also download the report in **CSV / Excel** format.

---

##  Features
- Add friends and their total expenses
- Automatic calculation of:
  - Total trip expense  
  - Per person share  
  - Who will **PAY** and who will **GET**
  - Exact transfers (**who pays whom**)  
- Download report:
  -  CSV
  -  Excel
- Reset all data with one click
